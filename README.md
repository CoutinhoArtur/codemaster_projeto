# 💻 Portfólio CodeMaster Dev

Projeto desenvolvido durante o curso técnico de **Desenvolvimento de Sistemas – SENAI**, com o objetivo de transformar uma página web estática em uma **interface interativa utilizando JavaScript Vanilla**.

O projeto simula o desenvolvimento de um **portfólio profissional de desenvolvedor**, adicionando funcionalidades interativas, animações e integração com formulário.

---

# 🎯 Objetivo do Projeto

Aplicar conceitos de **JavaScript e manipulação do DOM** para implementar funcionalidades em uma interface previamente construída com **HTML e CSS**.

Durante o desenvolvimento foram trabalhados:

- Manipulação do **DOM**
- **Eventos em JavaScript**
- **Funções**
- **Animações com JavaScript**
- **Interação com formulários**
- **Responsividade**
- **Versionamento com GitHub Desktop**

---

# 🛠 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Git
- GitHub Desktop
- VS Code
- Live Server

# 📌 Estado Inicial do Projeto

Inicialmente o projeto possuía apenas:

- Estrutura **HTML completa**
- Estilização em **CSS**
- Arquivo **JavaScript vazio**

A página carregava corretamente no navegador, porém **não possuía interatividade**.

Problemas iniciais:

❌ Menu mobile não funcionava  
❌ Navegação não destacava a seção ativa  
❌ Sem animações na página  
❌ Carrossel não funcional  
❌ Formulário sem tratamento em JavaScript
